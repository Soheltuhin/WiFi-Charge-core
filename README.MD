# WiFi-Charge Core 🚀

**WiFi-Charge** is an innovative mobile application designed to empower users by allowing them to carry forward their unused home broadband data for use while outdoors. 

### 🛠 Project Objectives:
* **Data Efficiency:** Minimize data waste by maximizing home internet utilization.
* **Seamless Connectivity:** One-tap access to your home data from anywhere.
* **Secure Authentication:** User login via Mobile Number and Firebase OTP verification.
